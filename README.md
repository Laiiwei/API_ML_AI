## 二、核心价值（最小可行性产品）
本产品旨在为学龄前儿童提供更好的英汉双语口语教学服务，通过趣味的方式、标准的语音来引导学龄前儿童进行口语学习，同时减轻父母教学的压力，保障教学质量。用户可根据自身学习情况选择相对应的学习内容，并跟随软件所提供的教学方案进行口语学习。
